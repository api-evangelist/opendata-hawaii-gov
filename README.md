# Hawaii Open Data Portal (opendata-hawaii-gov)

Hawaii Open Data Portal is a state government open-data portal for United States running **CKAN**, exposing a standardized catalog API over ~976 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/opendata-hawaii-gov/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=opendata-hawaii-gov-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.11

## API
- **Hawaii Open Data Portal CKAN Action API** — base `https://opendata.hawaii.gov/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://opendata.hawaii.gov)
- Live endpoint verified: `https://opendata.hawaii.gov/api/3/action/status_show` · ~976 datasets
